What is it?

This is a simple program in which you can add certain products to the basket and proceed to checkout, just like any other online purchasing sites. I also developed the program so it would apply promotional offers and could eventually be run through a database. By doing this task I was able to develope many skills and understand more aspects of programming (especially in ruby)

  • Understanding the importance of Object Orientated Design
  • Improved development of classes and functions
  • Understanding how ruby databases work and why they can be useful
