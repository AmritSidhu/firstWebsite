Welcome to my battlehips game!

Gameplay

That player begins the game by typing out the letter and number position from the grid (i.e. “A9”), and the game responds by determining if the called shot was a “hit” or a “miss.” Hits occur when the called shot hits any part of an enemy vessel. A miss occurs when the shot misses all enemy vessels.

  • If a vessel was hit, the board displays "@" and says “Hit”
  • If the geuss was a miss, the board displays "o" and says “Miss”
  • The first player to sink the entire enemy fleet wins the game.
Rules

The important part of this project was to help develope my Object Orientated Design and understand the importance of safegaurding. I put many safegaurds within this project to make sure that the user cannot bring up errors or break the game. Some of the rules I set are listed below;

  • Ships can be placed in a horizontal or vertical position anywhere within the grid
  • No part of the ship can be out of the grid, nor can they overlap in any position with another ship.
  • Ship placement is permanent and cannot be changed once the game has started.
